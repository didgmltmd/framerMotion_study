
# 🎞️ Framer Motion Study

프론트엔드에서 **자연스러운 모션과 인터랙션**을 연습/정리하기 위한 개인 스터디 레포지토리입니다.   
작은 예제들을 통해 재사용 가능한 패턴을 쌓아가고, 실제 프로젝트에 바로 복붙할 수 있는 수준의 코드 스니펫을 목표로 합니다.  

---

## 프로젝트 개요
- **목적**: Framer Motion의 핵심 API를 예제로 익히고, 자주 쓰는 모션 패턴을 컴포넌트화

---


## 🛠️ 기술 스택

* **React** (18+)
* **Framer Motion** (^11)
* 번들러: Vite
* 언어: JavaScript
